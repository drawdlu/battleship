# BattleShip Game

A battleship game built on vanilla javascript.

## Features

- game start options
  - player vs player
  - player vs computer
  - player names
  - optional fleet damage indication
  - optional ship reveal button
- ship deployment
  - drag and drop
  - rotate
  - randomize
- computer player
  - guesses randomly between available coordinates
  - finds a path / direction when a ship is hit
- fleet damage indication
  - when turned on it shows which of the five ships is hit and how many times it has been hit
- ship reveal button (player vs player)
  - allows current player to see their ships on board during button hold
- restart / play again
  - allows players to restart at any point in the game

## Technologies Used

- HTML
- CSS
- Javascript
- Weback
- Babel
- Jest
